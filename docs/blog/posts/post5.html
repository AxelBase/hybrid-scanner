Batch Scanning Workflow Guide

Batch Scanning Workflow Guide

Published: December 24, 2025

The AxelBase Hybrid Scanner includes a powerful batch mode designed for situations where multiple hybrid codes need to be processed efficiently. This feature allows users to decrypt several secrets in sequence without restarting the scanning process each time.

When batch mode is enabled, the scanner collects all successfully decrypted messages in a list rather than replacing the previous result. Users can set a target number of scans, and the system will automatically pause once that count is reached, preventing accidental over-scanning.

The workflow begins with activating batch mode and optionally specifying the desired number of items. Scanning then proceeds normally: each valid hybrid pair presented to the camera adds its decrypted secret to the growing collection. Progress is visible in real time.

Effective Batch Strategies

Preparation is key for smooth batch operations. Organizing physical documents in sequence reduces handling time between scans. Good lighting and consistent camera distance help maintain reliable detection throughout the session.

Practical Tips

  • Set realistic target counts based on document volume
  • Use export features mid-session if needed
  • Pause and resume without losing collected results
  • Review all items before final export

Mid-Session Management

Even before reaching the target or completing the batch, users can copy individual results or export the entire current collection as a text file. This flexibility supports workflows where partial results need immediate use or verification.

Batch mode maintains the same high security standards as single scanning, with all decryption occurring locally and no data transmitted externally.

FAQ

Does batch mode affect scanning speed?

No. Detection and decryption performance remains identical to single mode.

Can I change the target count during scanning?

Yes. Adjusting the target updates the auto-stop behavior immediately.

What happens if I scan the same code twice?

Duplicate identical codes are ignored to prevent repeated entries.

Batch mode transforms the hybrid scanner into an efficient tool for processing volume while preserving security and privacy.
